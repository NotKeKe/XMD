## embed_set_channel

### description

```
## 簡介
將此頻道設定為**推文下載頻道**後，你可以直接在這裡放入 X(Twitter) 的連結，而我就會直接將該貼文的圖片、影片下載至你在 config.toml 所設定的路徑。
同時也會將連結傳到這裡。

## 當前狀態
- 該頻道是否為**推文下載頻道**: {enable}
```

## embed_help

### description

```
## 指令列表

- `/set_channel`: **設定下載頻道**。管理此頻道的自動下載功能。
- `/download <url>`: **手動下載**。輸入 X (Twitter) 連結來手動下載媒體。
- `/my_id`: **查詢 ID**。取得你在 Discord 的使用者 ID。
- `/help`: **指令幫助**。顯示此說明訊息。

## 功能說明

1. **自動下載**：使用 `/set_channel` 將頻道設定為「推文下載頻道」後，只要在該頻道貼上 X 的連結，機器人就會自動解析並下載該貼文的所有圖片與影片。
2. **手動下載**：使用 `/download` 並提供連結，可以預覽貼文內容並選擇下載個別媒體或全部媒體。
```
